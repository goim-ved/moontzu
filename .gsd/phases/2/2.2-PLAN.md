# Plan 2.2: Tenant Registration and Onboarding

## Objective
Create the interface and backend logic for authenticated users to create a new tenant (business) on the platform, automatically assigning them as the owner in the multi-tenant database schema.

## Context
- .gsd/SPEC.md
- src/app/login/page.tsx
