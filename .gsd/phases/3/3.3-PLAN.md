## Objective
Convert the previously built Branded Landing Page (`src/app/[domain]/page.tsx`) into a fully functional public booking interface where anonymous users can traverse services and submit booking requests.

## Context
- src/app/[domain]/page.tsx
- .gsd/SPEC.md
